# Sky Diving
 Skydiving is 3D VR Simulator
 
## Want to Contribute to code?? 
Anyone can contribute to this project by **creating new maps** and fixing some **bugs**.

**Tools you will need**
1. Unity 3D
2. Blender to create 3D Models (optional)
